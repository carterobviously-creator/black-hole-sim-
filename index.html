Mini‑GW Web Song Studio

No external files required. This single static page generates music from your prompt using a deterministic in‑browser engine and synthesizes WAV audio you can download. It also includes a simple Grok‑like text reply generator (local, not a hosted model).

Status: idle
This page runs entirely in your browser. No server, no uploads, no external models required.
When generation finishes, a WAV download link appears in the console. The page also prints a short Grok‑style text response describing the song.
Drop this file into a GitHub Pages repo or open locally. Works offline in modern browsers (Chrome, Edge, Firefox).
